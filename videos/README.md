# Videos

Rendered animations and raw movie artefacts live here so they stay out of the source tree:

- `generated/` – exports created by experiments or scripts (git-ignored).
- `presentations/` – slide-related renders, including large `partial_movie_files` directories (git-ignored).
- `scripts/` – Manim or auxiliary Python files that generate the animations.

Commit scripts and metadata; keep binary outputs in the ignored subfolders.
