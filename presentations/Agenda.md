## General Presentation Notes
**Design Consistency**
- 16:9 widescreen aspect ratio  
- Two-tone color scheme (neutral + accent)  
- Sans-serif font family (e.g., Helvetica or Lato)  

**Technical Workflow**
- Export Manim animations as `.mp4` or `.mov`  
- Embed using Keynote’s “Movie” tool  
- Test transitions for timing and smooth playback
